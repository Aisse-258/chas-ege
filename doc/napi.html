Справка по NApi

Предполагается, что API получено способом, описанным выше

Переменные

Название Пояснение Аналог в старом API
NApi.info.VERSION Версия API
NApi.state.debug_mode Флаг режима отладки
NApi.task.text Текст вопроса window.vopr.txt
NApi.task.answers Правильные ответы window.vopr.ver
NApi.task.tags Теги тем window.vopr.kat
NApi.task.check_answer Функция проверки ответа
Прототип:function(answer) { ... }		 window.vopr.kat
NApi.task.draw Функция отрисовки window.vopr.dey
NApi.task.reset Сбросить все переменные NApi.task в исходные значения window.vopr.podg()

Функции

Название Пояснение Пример Аналог в старом API
NApi.Linfo Вывести в консоль информацию NApi.Linfo("Hello, world"); console.log("Hello, world");
NApi.Lmsg Вывести в консоль сообщене) NApi.Lmsg("Got Hello, world"); console.log("Hello, world");
NApi.Lwarn Вывести в консоль предупреждение NApi.Lwarn("File not found"); console.log("Warn: Hello, world");
NApi.Lerr Вывести в консоль информацию об ошибке NApi.Lerr("Failed to load file"); console.log("Error: Hello, world");
NApi.Ldebug Вывести в консоль отладочную информацию.
Выводиться только при NApi.state.debug_mode = true		 NApi.Ldebug("var foo = " + foo);
