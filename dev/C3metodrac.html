Метод рационализации

  • $\mathrm{sgn}(\log_a x)=\mathrm{sgn} ( (a-1) (x-1) )$
    Следствие:
  • $ \mathrm{sgn}(\log_a x - \log_a y)= \mathrm{sgn}(\log_a \frac{x}{y})= \mathrm{sgn} ( (a-1) (\frac{x}{y}-1))= \mathrm{sgn} ( (a-1) (x-y) ) $ Последний переход возможен, т. к. по ОДЗ логарифма $\mathrm{sgn}(y)=1$
  • $\mathrm{sgn}(a^x - a^y)=\mathrm{sgn} ( (a-1) (x-y))$
    Следствие:

Пример применения:

Неравнество $$\log_{x+7}(x^2-x+3) \leq \log_{x+7}(x-5)$$ равносильно неравенству $$\log_{x+7}(x^2-x+3) - \log_{x+7}(x-5) \leq 0$$ а следовательно, и неравенству $$((x+7)-1)((x^2-x+3) - (x-5)) \leq 0$$ (с учётом ОДЗ!)

Неравнество $$(x+7)^{x^2-x+3} \leq (x+7)^{x-5}$$ равносильно неравенству $$(x+7)^{x^2-x+3} - (x+7)^{x-5} \leq 0$$ а следовательно, и неравенству $$((x+7)-1)((x^2-x+3) - (x-5)) \leq 0$$ (с учётом ОДЗ!)
